 🎓 Student Placement Prediction using Logistic Regression

A Machine Learning project that predicts whether a student will be Placed or Not Placed based on academic performance and extracurricular activities. The model is built using **Logistic Regression** and deployed using **Streamlit**.
This has been made by me to experiment whether I am able to analyse the data properly or not.


📌 Project Overview

This project uses six important student-related parameters to predict placement status.

Input Features:
- CGPA
- Number of Internships
- Number of Projects
- Aptitude Test Score
- Soft Skills Rating
- Extracurricular Activities

Target:
- 1 → Placed
- 0 → Not Placed

---

🚀 Model

Algorithm Used:
- Logistic Regression

Model Accuracy:
- 75%

The model was trained on the dataset after preprocessing and evaluated using a train-test split, achieving an accuracy of 75%.

▶️ How to Run

Clone the repository

git clone https://github.com/MehulSahai410/ML_project_logreg.git

Move into the project directory

cd ML_project_logreg

Install dependencies

pip install -r requirements.txt

Run the application

streamlit run app.py
